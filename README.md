## 使用net包实现的简单端口扫描器
***
### 使用方法 (以命令行工具为例)
由于这是一个开发练手工具，我们假设它会以一个简单的可执行文件运行。
### 1. 构建 (Build)
```
切换到 main.go 所在目录 (例如 cmd/scanner)
go build -o portscan .
```

### 2. 运行示例
```
扫描目标 IP 的常用端口 (例如 1 到 1000)
语法: ./portscan <目标IP/域名> <起始端口> <结束端口>
./portscan 192.168.1.1 1 1000
扫描本机常用 Web 端口
./portscan localhost 80 8080`
```


### 开发者提示
#### 如果您对并发性能优化感兴趣，建议您尝试以下改进方向：
**1. Context 控制：引入 context 包，实现整个扫描任务的优雅取消和超时控制。**
**2. 结果输出：将扫描结果输出到文件，而不是直接打印到控制台。**
**3. 协议识别：在端口开放后，尝试发送简单的 HTTP/SSH 握手包，识别目标端口运行的服务类型。**
